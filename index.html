Flagship Project

Parqcel — Desktop workbench for Parquet, CSV, and Excel

Parqcel is a PyQt6 + Polars application built for fast, reliable data inspection and transformation. It supports pagination for large datasets, inline editing with undo/redo, data typing, rich filtering/statistics, and optional AI-assisted code generation with validation before execution.

  • Large-file friendly: pagination and Parquet-first workflows keep exploration responsive.
  • Safe editing: reversible edits, column operations, and cleaner export paths.
  • Useful analytics: multi-sort, profiling, filtering, and optional PCA/UMAP workflows.
  • AI optionality: natural-language to Polars code with validation checks.
